🐦 Flappy Bird Clone

A simple yet addictive Flappy Bird-inspired browser game built using HTML, CSS, and JavaScript. Control the bird, avoid obstacles, and try to achieve the highest score!

🚀 Features
🐦 Smooth bird movement with gravity physics
🎮 Tap/keypress controls to fly
🚧 Moving obstacles (pipes)
💥 Collision detection (game over)
🎯 Score tracking system
🔄 Restart functionality
🎨 Clean and minimal UI
🛠️ Tech Stack
HTML5 – Game structure
CSS3 – Styling & animations
JavaScript (Vanilla) – Game mechanics

🎮 Controls
⌨️ Spacebar / Click – Make the bird fly
❌ Avoid hitting pipes or ground
